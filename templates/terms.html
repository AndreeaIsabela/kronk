Commands

The Bot provides the following slash commands:

  • /pet-buff — starts a 2-hour pet buff 🐯💥 timer for yourself or specified players.
  • /pet-buff-cancel — cancels an active pet buff timer.
  • /set-rally-leaders — sets the server's rally leaders list.
  • /rally-leaders — displays the current rally leaders list.
  • /anonymous — posts an anonymous message to a chosen channel; optionally includes instructions for anonymous replies.
  • /set-event — schedules a one-time or recurring event notification in a channel.
  • /list-events — lists all scheduled events in the server and their details.
  • /cancel-event — cancels a scheduled event by name.
